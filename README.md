# cafe-sales-tool
VBAで作ったカフェの売上管理ツールです。

## 使い方v
G,1プルダウンで商品を選択。<br>
G,2には数量を入れてください。<br>
合計を算出します。<br>

## 機能
・商品入力<br>
・数量入力<br>
・合計計算<br>
・商品別売上<br>
