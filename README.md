# cafe-sales-tool
VBAで作ったカフェの売上管理ツールです。

## 使い方
G,1プルダウンで商品を選択。<br>
G,2には数量を入れてください。<br>
合計を算出します。<br>

## スクリーンショット
![画面](screenshot.jpg)

## 機能
・商品入力<br>
・数量入力<br>
・合計計算<br>
・商品別売上<br>
・売上グラフ表示<br>
・売上データ削除<br>

## 使用技術
・Excel<br>
・VBA<br>
・WorksheetFunction<br>
・SUM/SUMIF<br>
・グラフオブジェクト（ChartObject）<br>

## 開発の目的
VBAの使用方法を覚える為に資格勉強を取り組み、その過程から実践的なスキルを習得しようと開発を始めました。小さな機能を積み上げてツールを完成させていきたいです。
